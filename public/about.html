The goal of OpenShapes is to enable in-the-wild image synthesis and manipulation using a simple paint-brush and a drag-and-drop tool. This work appeared at the Conference of Computer Vision and Pattern Recognition (CVPR) in 2019. The source code for this project is available on GitHub. This code does not require any expensive hardware, and one may efficiently run it on their regular machines for customizable outputs. Our system currently uses only 120,000 images from COCO. We are in process of increasing it to an order more of data for better results.

Harsha and Ankita designed the front-end, and integrated it with the back-end. For technical details, please see:


A. Bansal, Y. Sheikh, D. Ramanan
Shapes and Context: In-the-wild Image Synthesis & Manipulation.
In CVPR, 2019.
